# Подсказки к командной строке

команда смены директории
```sh
cd c:\folder_name
```

Команда отображения текущей директории.
MacOs, Linux
```sh
pwd
```
Листинг текущей директории
Windows:
```sh
dir
```
MacOs, Linux:
```sh
ls
```

Удаление файла в Windows:
```sh
del <filename>
```
в MacOs, Linux:
```sh
rm <filename>
```
